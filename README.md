# UFO Desktop

macOSデスクトップ上にUFOキャラクターをふわふわ浮遊表示するアプリ。

## セットアップ

```bash
uv sync
uv run python ufo_app.py
```

## 操作方法

| 操作 | 動作 |
|------|------|
| シングルクリック | 浮遊の停止 / 再開 |
| ダブルクリック | スクリーンショット起動 |
| 停止中にドラッグ | UFOを好きな位置へ移動 |
| メニューバー 🛸 → UFO 停止/起動 | 浮遊の停止 / 再開 |
| メニューバー 🛸 → 終了 | アプリ終了 |

- 停止中は🛸💤に変わります
- スクリーンショットは `ufocapture/` フォルダに保存されます
- ダブルクリック後、範囲選択してキャプチャ（Escでキャンセル）

## 動作環境

- macOS 12 (Monterey) 以降
- Python 3.9 以降
